# React Custom Hooks Demo

Build your own chat UI using `useChat`, `useSessions`, and `ClaudeProvider`.

## Run

```bash
node examples/react-custom-hooks/server.js
```

Open http://localhost:4011

## What This Demonstrates

- Full control over UI layout and styling
- Sidebar session list with create/resume
- Streaming messages via WebSocket
- Tool-use display inline
- Config display in header
